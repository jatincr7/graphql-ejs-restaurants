# GraphQL & Server-Side Rendering Restaurant Listing

This is a simple Node.js application that demonstrates how to integrate GraphQL for data retrieval and perform server-side rendering using EJS templating. The application displays a list of restaurants along with their details.

## Features

- GraphQL API for retrieving restaurant data
- Server-side rendering using EJS template engine
- Basic HTML and CSS for styling

## Prerequisites

- Node.js and npm installed on your system
